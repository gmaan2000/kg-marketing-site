# K&G Design System

## Signature Typography

### The "Vanguard" / "Globally" Style
This is the projects signature "Glitch/3D" text style used for emphasis.

**CSS Class:** `.accent-text`

**Visual Traits:**
- Font: Inter (Weights 900)
- Effect: 3D Layered Text Shadow (Green `#00cc66`)
- Transform: Skewed -5deg
- Stroke: 1px Black

**How to use:**
Simply wrap any text in a span with this class:
```html
<h1>Join the <span class="accent-text">Revolution</span></h1>
```

## Color Palette
- **Neon Green**: `#00ff88` (Main Accent)
- **Deep Green**: `#00cc66` (Shadows)
- **Neon Blue**: `#00bbff` (Secondary Accent)
- **Dark BG**: `#0a0a0a`

## UI Elements
- **Buttons**: Use `.btn-glow` for the main call-to-action (pulsing green glow).
- **Glassmorphism**: Use `.glass` or `.glass-nav` for transparent backgrounds.
